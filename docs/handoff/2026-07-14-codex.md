# Codex handoff — winget distribution retired (2026-07-14)

## Product decision

The user retired the planned `microsoft/winget-pkgs` submission on 2026-07-14.
Do **not** submit a winget PR, bump the local winget manifests, run winget
validation as release work, or advertise `winget install NotNull92.Hera`.

No public winget PR was ever opened. The existing `packaging/winget/` files are
kept solely as a historical, locally validated artifact.

## Rationale

Hera is a Godot addon with a companion CLI. winget would distribute only the
CLI, while the supported addon path is the Godot Asset Store or the addon
release ZIP. Treating a general Windows catalog as an active distribution
channel would misrepresent the product's Godot-centered installation flow.

Reconsidering winget requires a new explicit user product decision. The
presence of the archived manifests, a previously successful local validation,
or an unfilled roadmap item is not authorization to reopen it.

## Summary

Low-token CLI and Godot editor addon that lets AI coding agents inspect, control, and verify a live Godot 4.7+ editor through compact shell commands.

## Description

Hera Agent Godot is a Godot 4.7+ editor addon plus companion CLI for agent-driven development. It exposes compact commands for live editor status, scene/node inspection, editor selection, script and resource workflows, signal wiring, runtime game inspection, screenshots, diagnostics, and automated QA scenarios.

The addon runs inside the real Godot editor and communicates over localhost, so agents can act on the actual open scene, run the game, read errors, capture screenshots, and confirm that changes worked instead of guessing from files alone. Outputs are compact JSON by default to keep agent context usage low.
